# 字段映射与匹配草案

## Excel 参考字段

| Excel字段 | 用途 | 备注 |
|---|---|---|
| 序号 | 日志追踪/目录命名 | 建议保留 |
| 凭证编号 | 主检索键 | 高优先级 |
| 凭证日期 | 主检索键 | 标准化为 YYYY-MM-DD |
| 科目编号 | 辅助检索键 | 与科目名称配合 |
| 科目名称 | 辅助检索键 | 可能包含层级文本 |
| 对方科目 | 辅助检索键 | 可能用于二次确认 |
| 借方发生金额 | 主/辅检索键 | 与贷方发生金额二选一 |
| 贷方发生金额 | 主/辅检索键 | 与借方发生金额二选一 |
| 摘要 | 主/辅检索键 | 文本匹配时允许模糊/包含 |
| 附件索引号 | 附件确认/命名 | 可能有多个值 |
| 附件内容填写 | 附件确认 | 如：银行支付单、销售合同 |
| 抽凭名称 | 分目录/筛选 | 可选 |
| 关联抽凭名称 | 分目录/筛选 | 可选 |

## 建议匹配顺序

1. 先用 凭证编号/凭证日期 检索
2. 再用 金额 + 摘要 过滤
3. 再用 科目编号/科目名称/对方科目 二次确认
4. 如果站内天然以附件索引号检索，优先切换为附件索引号检索
